# 변수(Variable)란?

수학적인 의미로 변할 수 있는 숫자이나 프로그래밍에서는 이름을 할당받고 데이터를 저장하는 메모리 공간입니다.

- 메모리: 데이터를 위한 보관함 (보관함 크기 동일)
- 변수: 각 데이터 보관함의 이름 (이름을 통해 재활용)

## 변수 사용법:

1. 보관함 확보(선언 declaration)

```jsx
let 변수이름;
```

2. 보관함에 데이터 저장(할당 assignment)

```jsx
age = 12;
```

'='의 의미는 할당되다, 저장되다./ 같다가 아님

보관함 확보와 데어터 저장을 같이 할 수 있음

```jsx
let age = 12;
```

실생활에 비슷한 개념을 찾아보면 헬스장에서 사물함과 가장 유사한거 같습니다.
운동복이나 개인소지품을 보관하기 위한 보관함(헬스장 전체 사물함=메모리)이고 사물함 크기는 다 동일합니다.

회원권을 결제하거나 사물함 이용권을 구매하여 자기만의 사물함을 가지고(보관함 확보) 자기만의 사물함에 소지품을 보관합니다.(데이터 저장)
현실이랑 가장 다른 점은 보관한 데이터를 보관함의 이름 즉 변수를 통해 재활용해서 사용할 수 있다는 것입니다.

자바스크립트에서 변수명을 선언할 때 주로 사용하는 공식은 loWerCamelCase 입니다.
첫단어 첫글자를 소문자로 표기하고 다음단어 첫글자부터는 대문자로 표기합니다

- CamelCase
  두개 단어 이상 모인 합성어의 각 단어의 첫 글자를 대문자로 표기하여 쌍봉낙타 등처럼 혹이 튀어나왔다 들어가는 모양이랑 비슷해서 CamelCase라고 불립니다.

1. lowerCamelCase: 첫단어 첫글자 소문자 + 그외 나머지 단어 대문자 (공백없음)
   예) userData, finalDate, addResult
2. UpperCamelCase: 모든단어 첫글자 대문자 (공백없음)
   예) UserData, FinalDate, AddResult
